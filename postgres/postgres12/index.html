    еңЁpostgres 12 зүҲжң¬дёӯж–°еўһдёҖдёӘжҝҖжҙ»д»Һеә“дёәдё»еә“зҡ„ж–№ејҸгҖӮpg_promote еҮҪж•°пјҢзӣёжҜ”еҺҹжңүзҡ„дёӨз§Қж–№ејҸпјҢиҝҷз§Қж–№жі•зҡ„дјҳзӮ№еңЁдәҺдёҚйңҖиҰҒзҷ»йҷҶеҲ°е®һдҪ“жңәдёҠпјҢеҸҜиҝңзЁӢйҖҡиҝҮsqlиҝӣиЎҢж“ҚдҪңгҖӮ pg_promote() еҮҪж•°жңүдёӨдёӘеҸӮж•°:

    • wait: иЎЁзӨәжҳҜеҗҰзӯүеҫ…еӨҮеә“зҡ„ promotion е®ҢжҲҗжҲ–иҖ… wait_seconds з§’д№ӢеҗҺиҝ”еӣһжҲҗеҠҹпјҢй»ҳи®ӨеҖјдёә trueгҖӮ
    • wait_seconds: зӯүеҫ…ж—¶й—ҙпјҢеҚ•дҪҚз§’пјҢй»ҳи®Ө 60

DROP TABLE measurement_y2006m02;
--- йҖҡеёёж–№ејҸпјҢе°ҶеҲҶеҢәиЎЁи„ұзҰ»еҮәдё»иЎЁпјҢз„¶еҗҺеңЁеҜ№еҲҶеҢәиЎЁиҝӣиЎҢж“ҚдҪң
ALTER TABLE measurement DETACH PARTITION measurement_y2006m02;

--- е°Ҷе·ІжңүиЎЁдҪңдёәеҲҶеҢәиЎЁеҠ е…ҘеҲ°дё»иЎЁдёӯ, зӣҙжҺҘеҠ е…Ҙдјҡй”Ғдё»иЎЁ
ALTER TABLE measurement ATTACH PARTITION measurement_y2008m02
    FOR VALUES FROM ('2008-02-01') TO ('2008-03-01' );

--- йҖҡеёёеҒҡжі•пјҢеҜ№йңҖиҰҒеҜ№еҠ е…Ҙзҡ„еҲҶеҢәиЎЁеҠ жЈҖжҹҘзәҰжқҹпјҢз„¶еҗҺеңЁе°ҶеҲҶеҢәиЎЁеҠ е…ҘеҲ°дё»иЎЁдёӯ

ALTER TABLE measurement_y2008m02 ADD CONSTRAINT y2008m02
   CHECK ( logdate >= DATE '2008-02-01' AND logdate < DATE '2008-03-01' );

ALTER TABLE measurement ATTACH PARTITION measurement_y2008m02
    FOR VALUES FROM ('2008-02-01') TO ('2008-03-01' );

    йҖҡеёёжғ…еҶөдёӢеҰӮжһңеҜ№дёҖеј иЎЁеҠ е…Ҙзҙўеј•дјҡе өеЎһиҜҘиЎЁзҡ„dmlж“ҚдҪңпјҢзү№еҲ«жҳҜеҜ№дёҖеј еӨ§иЎЁзҡ„ж“ҚдҪңгҖӮ

    й»ҳи®Өжғ…еҶөдёӢпјҢеҰӮжһңеңЁдё»иЎЁдёӯеҠ е…ҘдёҖдёӘзҙўеј•пјҢиҜҘзҙўеј•д№ҹдјҡеҠ е…ҘеҲ°зҙўеј•зҡ„еӯҗиЎЁдёӯпјҢж— и®әжҳҜзҺ°жңүзҡ„еӯҗиЎЁиҝҳжҳҜе°ҶжқҘж–°еҠ е…Ҙзҡ„еӯ—иЎЁгҖӮиҝҷж ·жһҒеӨ§зҡ„ж–№дҫҝдәҶеҜ№еҲҶеҢәиЎЁзҡ„з»ҙжҠӨгҖӮзӣ®еүҚеңЁеҲҶеҢәиЎЁдёҠе»әз«Ӣзҙўеј•ж—¶дёҚж”ҜжҢҒCONCURRENTLY

    дҪҶжҳҜиҖғиҷ‘зҡ„еҜ№еңЁзәҝдёҡеҠЎзҡ„еҪұе“ҚпјҢеңЁеҲҶеҢәиЎЁдёӯе»әи®®зҡ„ж“ҚдҪңжөҒзЁӢгҖӮ

    йҰ–е…ҲеңЁдё»иЎЁдёҠдҪҝз”Ё create index on only иҜӯеҸҘеҲӣе»әзҙўеј•пјҢз„¶еҗҺеҲҶеҲ«еңЁеӯҗиЎЁдёҠеҲӣе»әзҙўеј•гҖӮеҪ“жүҖжңүеӯҗиЎЁдёҠзҡ„зҙўеј•йғҪе»әз«Ӣе®ҢжҜ•еҗҺдё»иЎЁдёҠзҡ„зҙўеј•иў«жҝҖжҙ»гҖӮ
